Pattern-matching on arrays

You can also pattern match on arrays: 

let has_size_two = function
  | [| _; _ |] -> true
  | _          -> false
And you can mix all the kind of values: 
let f = function
  | []                 -> failwith "empty list"
  | [| _; (_, x) |]::_ -> x
  | _                  -> failwith "the first array should be of size two"
