Descrição da Tarefa - Demonstrar vs Refutar

Vocês receberam um conjunto de problemas cuja tarefa envolve Demonstrar ou Refutar. Nós adquirir as habilidades necessárias para aplicar as regras do jogo de demonstração em Dedução Natural, bem como o jogo semântico de Refutação, a árvore de verificação pela Teoria Semântica.

Ao acessar sua conta do ProofWeb, vocês verão uma lista de exercícios :


aval_nivel1_0_ND.v
aval_nivel1_0_SEM.v

Para cada par destes arquivos, apenas um deles pode ser resolvido corretamente.

O arquivo aval_nivel1_0_ND.v está formulado no Template de dedução natural e equivale a demonstrar que P1,P2,P3 ⊢ C . 


Reset Initial.	
Require Import ProofWeb.
Parameter A B C: Prop.
Hypothesis P0 : ( C -> ( B /\ A ) ).
Hypothesis P1 : ( ( A -> A ) -> ( C <-> B ) ).
Theorem T0 : ( ( C \/ ( C -> A ) ) -> ( A -> C ) ).

Já o arquivo com aval_nivel1_0_ND.v está formulado no template da teoria Semântica e equivale a verificar que ⊭(P1∧P2∧P3)→C . 


Reset Initial.
Require Import Semantics.
Parameter A B C: Prop.
(* Hypothesis P0 : (v ?? A). *)
(* Hypothesis P1 : (v ?? B). *)
(* Hypothesis P2 : (v ?? C). *)
Theorem T0 : (v ||-/- ( ( ( C -> ( B /\ A ) ) /\ ( ( A -> A ) -> ( C <-> B ) ) ) -> ( ( C \/ ( C -> A ) ) -> ( A -> C ) ) ) ).

ATENÇÃO! Apenas uma dessas duas formulações do problema pode e precisa ser resolvido.
